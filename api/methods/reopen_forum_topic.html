重新打开论坛话题#

Returns: bool

class aiogram.methods.reopen_forum_topic.ReopenForumTopic(*, chat_id: int | str, message_thread_id: int, **extra_data: Any)[源代码]#

使用此方法在超级组群聊中重新打开已关闭的话题。 机器人必须是聊天中的管理员才能这样做，并且必须具有*can_manage_topics*管理员权限，除非它是话题的创建者。 成功时返回:code:True。

Source: https://core.telegram.org/bots/api#reopenforumtopic

chat_id: int | str#

目标聊天的唯一标识符或目标超级群的用户名（格式为:code:@supergroupusername）

message_thread_id: int#

论坛主题目标消息线程的唯一标识符

用法#

作为机器人方法#

result: bool = await bot.reopen_forum_topic(...)

方法作为对象#

导入：

  • from aiogram.methods.reopen_forum_topic import ReopenForumTopic

  • alias: from aiogram.methods import ReopenForumTopic

使用特定的机器人#

result: bool = await bot(ReopenForumTopic(...))

在处理程序中作为 Webhook 回复#

return ReopenForumTopic(...)
