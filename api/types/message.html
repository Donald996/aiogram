此对象表示一条消息。

来源： https://core.telegram.org/bots/api#message

message_id: int#

此聊天中的唯一消息标识符

date: DateTime#

以Unix时间发送消息的日期。它始终是一个正数，表示有效日期。

chat: Chat#

聊天消息属于

message_thread_id: int | None#

可选。消息所属的消息线程的唯一标识符;仅用于超级组

from_user: User | None#

可选。消息的发送者;对于发送到通道的消息为空。为了向后兼容，如果消息是代表聊天发送的，则该字段在非频道聊天中包含伪造的发件人用户。

sender_chat: Chat | None#

可选。消息的发件人，代表聊天发送。例如，通道本身用于通道帖子，超级组本身用于来自匿名组管理员的消息，链接通道用于自动转发到讨论组的消息。为了向后兼容，如果消息是代表聊天发送的，则字段* from *在非频道聊天中包含伪造的发件人用户。

forward_origin: MessageOriginUser | MessageOriginHiddenUser | MessageOriginChat | MessageOriginChannel | None#

可选。有关转发消息的原始消息的信息

is_topic_message: bool | None#

可选。：代码： ‘True’，如果消息发送到论坛主题

is_automatic_forward: bool | None#

可选。：代码：“true”，如果消息是自动转发到连接的讨论组的频道帖子

reply_to_message: Message | None#

可选。对于同一聊天和消息线程中的回复，原始消息。请注意，此字段中的消息对象将不包含其他* reply_to_message *字段，即使它本身是一个回复。

external_reply: ExternalReplyInfo | None#

可选。有关正在回复的消息的信息，可能来自其他聊天或论坛主题

quote: TextQuote | None#

可选。对于引用原始消息一部分的回复，消息的引用部分

via_bot: User | None#

可选. Bot，通过它发送消息

edit_date: int | None#

可选。在Unix时间中最后编辑消息的日期

has_protected_content: bool | None#

可选. :代码： ‘True’，如果无法转发消息

media_group_id: str | None#

可选。此消息所属的媒体消息组的唯一标识符

author_signature: str | None#

可选。频道中消息的帖子作者签名，或匿名组管理员的自定义标题

text: str | None#

可选。对于文本消息，消息的实际UTF-8文本

entities: List[MessageEntity] | None#

可选。对于文本消息，文本中出现的特殊实体，如用户名，url，bot命令等

link_preview_options: LinkPreviewOptions | None#

可选。用于消息的链接预览生成的选项，如果它是文本消息和链接预览选项已更改

animation: Animation | None#

可选。消息是一个动画，有关动画的信息。为了向后兼容，当设置此字段时，还将设置* document *字段
