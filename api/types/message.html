这个对象表示一条信息。

Source: https://core.telegram.org/bots/api#message

message_id: int#

该聊天内的唯一消息标识符

date: DateTime#

消息发送的Unix时间。它总是一个正数，代表一个有效的日期。

chat: Chat#

消息所属的聊天

message_thread_id: int | None#

可选。消息所属的消息线程的唯一标识符；仅适用于超级群组

from_user: User | None#

可选. 消息发送者；对于发送到频道的消息为空。为了向后兼容，如果消息是代表聊天发送的，在非频道聊天中该字段包含一个虚假的发送者用户。

sender_chat: Chat | None#

可选. 以聊天发送方的名义发送消息的发送者。例如，对于频道帖子来说是频道本身，对于来自匿名群组管理员的消息来说是超级群本身，对于自动转发到讨论组的消息来说是相关联的频道。为了向后兼容性，在非频道聊天中，字段 from 中包含一个虚假的发送者用户，如果消息是代表聊天发送的。

sender_boost_count: int | None#

可选。如果信息发送者助力了聊天效果，则用户助力的次数

forward_origin: MessageOriginUser | MessageOriginHiddenUser | MessageOriginChat | MessageOriginChannel | None#

可选。关于转发消息的原始消息的信息

is_topic_message: bool | None#

Optional. True, 如果消息被发送到论坛主题

is_automatic_forward: bool | None#

可选的。True，如果消息是自动转发到连接的讨论组的频道帖子

reply_to_message: Message | None#

可选. 对于在同一聊天和消息线程中的回复，原始消息。请注意，此字段中的 Message 对象将不包含进一步的 reply_to_message 字段，即使它本身是一条回复。

external_reply: ExternalReplyInfo | None#

可选的。关于被回复的消息的信息，可能来自另一个聊天或论坛主题

quote: TextQuote | None#

可选的。对于引用原始消息的部分进行回复，被引用部分的消息

reply_to_story: Story | None#

选修的。对于故事的回复，原始故事

via_bot: User | None#

可选. 通过这条消息发送的机器人

edit_date: int | None#

可选。消息最后编辑的日期（Unix时间）

has_protected_content: bool | None#

可选的。 True, 如果消息不能转发

media_group_id: str | None#

可选. 此消息所属的媒体消息组的唯一标识符

author_signature: str | None#

可选。 在频道中发布的消息的作者签名，或匿名群组管理员的自定义标题

text: str | None#

可选的。对于文本消息，消息的实际 UTF-8 文本

entities: List[MessageEntity] | None#

可选的。对于文本消息，出现在文本中的用户名、URL、机器人命令等特殊实体

link_preview_options: LinkPreviewOptions | None#

可选。如果是文本消息并且链接预览选项已更改，则用于消息链接预览生成的选项

animation: Animation | None#

可选。消息是一个动画，关于动画的信息。为了向后兼容，当设置了此字段时，document 字段也将被设置

audio: Audio | None#

可选. 消息是一个音频文件，有关文件的信息

document: Document | None#

可选的。消息是一个普通文件，关于文件的信息

photo: List[PhotoSize] | None#

可选的。消息是一张照片，照片可用的尺寸

sticker: Sticker | None#

可选。消息是一个贴纸，有关贴纸的信息

story: Story | None#

可选。消息是转发的故事

video: Video | None#

可选。信息是一个视频，有关该视频的信息

video_note: VideoNote | None#

可选。消息是一个 视频笔记，关于视频消息的信息

voice: Voice | None#

可选。消息是一条语音消息，关于文件的信息

caption: str | None#

Optional. 动画，音频，文件，照片，视频或语音的标题

caption_entities: List[MessageEntity] | None#

可选。对于带有标题的消息，出现在标题中的用户名、URL、机器人命令等特殊实体
