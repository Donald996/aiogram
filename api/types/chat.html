聊天#

class aiogram.types.chat.Chat(*, id: int, type: str, title: str | None = None, username: str | None = None, first_name: str | None = None, last_name: str | None = None, is_forum: bool | None = None, photo: ChatPhoto | None = None, active_usernames: List[str] | None = None, available_reactions: List[ReactionTypeEmoji | ReactionTypeCustomEmoji] | None = None, accent_color_id: int | None = None, background_custom_emoji_id: str | None = None, profile_accent_color_id: int | None = None, profile_background_custom_emoji_id: str | None = None, emoji_status_custom_emoji_id: str | None = None, emoji_status_expiration_date: datetime | None = None, bio: str | None = None, has_private_forwards: bool | None = None, has_restricted_voice_and_video_messages: bool | None = None, join_to_send_messages: bool | None = None, join_by_request: bool | None = None, description: str | None = None, invite_link: str | None = None, pinned_message: Message | None = None, permissions: ChatPermissions | None = None, slow_mode_delay: int | None = None, unrestrict_boost_count: int | None = None, message_auto_delete_time: int | None = None, has_aggressive_anti_spam_enabled: bool | None = None, has_hidden_members: bool | None = None, has_protected_content: bool | None = None, has_visible_history: bool | None = None, sticker_set_name: str | None = None, can_set_sticker_set: bool | None = None, custom_emoji_sticker_set_name: str | None = None, linked_chat_id: int | None = None, location: ChatLocation | None = None, **extra_data: Any)[源代码]#

这个对象代表了一个聊天。

Source: https://core.telegram.org/bots/api#chat

id: int#

唯一聊天标识符。这个数字可能含有32位以上的有效位数，一些编程语言可能在解释时出现困难或潜在缺陷。但它最多有52位有效位数，因此使用带符号的64位整数或双精度浮点类型来存储这个标识符是安全的。

type: str#

聊天类型，可以是 private、 group、 supergroup 或 channel 其中之一

title: str | None#

可选. 超级群组、频道和群聊的标题

username: str | None#

可选. 用户名，在私聊、超级群组和频道中（如果可用）

first_name: str | None#

可选. 私聊中对方的名字

last_name: str | None#

可选. 私聊中对方的姓氏

is_forum: bool | None#

可选. True, 如果超级群聊是一个论坛（已启用 topics）

photo: ChatPhoto | None#

可选. 聊天照片。仅在 aiogram.methods.get_chat.GetChat 中返回。

active_usernames: List[str] | None#

可选. 如果非空，则返回所有私聊、超级群和频道的 活跃聊天用户名列表 。仅在 aiogram.methods.get_chat.GetChat 中返回。

available_reactions: List[ReactionTypeEmoji | ReactionTypeCustomEmoji] | None#

可选. 聊天中允许使用的可用反应列表。如果省略，则允许所有 表情反应 。仅在 aiogram.methods.get_chat.GetChat 中返回。

accent_color_id: int | None#

可选. 聊天名称和聊天头像、回复标题以及链接预览的强调颜色的标识符。有关更多详情，请参阅 强调颜色 。仅在 aiogram.methods.get_chat.GetChat 中返回。

background_custom_emoji_id: str | None#

可选. 由聊天选择的自定义表情符号标识符，用于回复标题和链接预览背景。仅在 aiogram.methods.get_chat.GetChat 中返回。

profile_accent_color_id: int | None#

可选. 聊天资料背景的重点颜色标识符。有关更多详细信息，请参阅 轮廓配色 。仅在 aiogram.methods.get_chat.GetChat 中返回。

profile_background_custom_emoji_id: str | None#

可选. 用户为其个人资料背景选择的自定义表情符号标识符。仅在 aiogram.methods.get_chat.GetChat 中返回。

emoji_status_custom_emoji_id: str | None#

可选. 在私人聊天中，自定义的表情符号标识聊天的表情状态或另一方的表情状态。仅在 aiogram.methods.get_chat.GetChat 中返回。

emoji_status_expiration_date: DateTime | None#

可选. 表示聊天或私人聊天中另一方的表情状态的过期日期，以Unix时间表示。仅在 aiogram.methods.get_chat.GetChat 中返回。

bio: str | None#

可选. 私人聊天中对方的简介。仅在 aiogram.methods.get_chat.GetChat 中返回。

has_private_forwards: bool | None#

可选. True，如果私聊对方的隐私设置允许只在与该用户的聊天中使用 tg://user?id=<user_id> 链接。仅在 aiogram.methods.get_chat.GetChat 中返回。

has_restricted_voice_and_video_messages: bool | None#

可选. True，如果对方的隐私设置限制在私人聊天中发送语音和视频笔记消息。仅在 aiogram.methods.get_chat.GetChat 中返回。

join_to_send_messages: bool | None#

可选. True，如果用户需要在能发送消息之前加入超级群组。仅在 aiogram.methods.get_chat.GetChat 中返回。

join_by_request: bool | None#

可选. True，如果所有直接加入超级群的用户都需要由超级群管理员批准。仅在 aiogram.methods.get_chat.GetChat 中返回。

description: str | None#

可选. 描述，用于群组、超级群组和频道聊天。仅在 aiogram.methods.get_chat.GetChat 中返回。

invite_link: str | None#

可选. 主要邀请链接，适用于群组、超级群组和频道聊天。仅在 aiogram.methods.get_chat.GetChat 中返回。

pinned_message: Message | None#

可选. 最近的置顶消息（按发送日期）。仅在 aiogram.methods.get_chat.GetChat 中返回。

permissions: ChatPermissions | None#

可选. 默认的群组和超级群组的聊天成员权限。仅在 aiogram.methods.get_chat.GetChat 中返回。

slow_mode_delay: int | None#

可选. 对于超级群组，每个无权限用户连续发送信息的最小允许延迟时间（秒）。仅在 aiogram.methods.get_chat.GetChat 中返回。

unrestrict_boost_count: int | None#

可选. 对于超级组，非管理员用户为忽略慢速模式和聊天权限而需要添加的最小提升数。仅在 aiogram.methods.get_chat.GetChat 中返回。

message_auto_delete_time: int | None#

可选. 向聊天发送的所有消息自动删除的时间，以秒为单位。仅在 aiogram.methods.get_chat.GetChat 中返回。

has_aggressive_anti_spam_enabled: bool | None#

可选. True，如果在超级组中启用了积极的反垃圾邮件检查。该字段仅适用于聊天管理员。仅在 aiogram.methods.get_chat.GetChat 中返回。

has_hidden_members: bool | None#

可选. True，如果非管理员只能获取聊天中的机器人和管理员列表。仅在 aiogram.methods.get_chat.GetChat 中返回。

has_protected_content: bool | None#

可选. True，如果来自聊天的消息不能转发到其他聊天。仅在 aiogram.methods.get_chat.GetChat 中返回。

has_visible_history: bool | None#

可选. True，如果新的聊天成员可以访问旧消息；仅聊天管理员可用。仅在 aiogram.methods.get_chat.GetChat 中返回。

sticker_set_name: str | None#

可选. 对于超级群组，群组贴纸集的名称。仅在 aiogram.methods.get_chat.GetChat 中返回。

can_set_sticker_set: bool | None#
